This function checks if the location is valid.

Validating location means finding a valid node in the generated map which is at a distance of less than 1 km from the given location.

Args:

graph (osmnx.graph): Generated graph of the source and destination city lat (float): location latitude long (float): location longitude

Returns:

osmnx.graph_node: nearest Valid node if the location is valid, None otherwise. bool: True if location is valid, False otherwise
